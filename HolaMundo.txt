Comando.


git init .... iniciar el repositorio.

git status ---- ver el status del repositori.

git add <file> ---- agregar archivos al rep.

git commit -m " mensaje " ----- agregar un mensaje detallado de los cambios realizados.

git config --global user.email o user.name " correo " ----- configurar datos del usuario.

git diff ---- ver los cambios

git checkout -- <file> .... descartar los cambios

git commit -a -m <file> .... saltar el git add <file> andes del commit -m

git log ---- ver los cambios realizados.

git show head ---- ultimo commit que se realizo

git add . ---- agregar todos los archivos al rep.

git reset "6 digitos del sha1 del commit".

git branch "nombre" .. crear rama

git checkout "nombre" .----- cambiar de rama.

git marge --no-ff "rama a combinar" .

git branch -d develop
